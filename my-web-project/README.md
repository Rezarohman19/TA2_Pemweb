Proyek ini merupakan hasil kerja saya dalam membangun sebuah website 
portofolio modern yang berfungsi sebagai media perkenalan diri sekaligus
tempat untuk menampilkan karya-karya yang telah saya buat selama
menempuh pendidikan di bidang Informatika.

Portofolio ini tidak hanya berfokus pada tampilan visual, tetapi juga pada
struktur kode yang bersih, terorganisir, dan mudah dikembangkan di masa
depan. Saya membuat proyek ini sebagai representasi dari kemampuan dan
minat saya di dunia frontend development serta sebagai langkah awal dalam
membangun identitas profesional di bidang teknologi informasi.

Website ini menampilkan beberapa bagian penting, seperti Tentang Saya,
Proyek, Keahlian, dan Kontak, yang masing-masing dirancang untuk
menggambarkan siapa saya, apa yang telah saya kerjakan, dan bagaimana cara
orang lain dapat berinteraksi dengan saya. Melalui portofolio ini, saya ingin
menunjukkan bahwa saya bukan hanya belajar untuk memahami teori
